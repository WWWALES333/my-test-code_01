## 1. 测试概述

### 1.1 测试目标

## 4. 验收标准

### 4.1 核心指标验收

| 指标 | 最低要求 | 验收方法 |
|------|----------|----------|
| 周报下载数量 | ≥20份/周 | 统计下载目录文件数 |
| 月报下载数量 | ≥8份/月 | 统计下载目录文件数 |
| 区域覆盖率 | ≥90% | 统计区域关键词覆盖 |
| 时间解析准确率 | ≥95% | 单元测试通过率 |
| 分类正确率 | 100% | 检查目录结构 |

### 4.2 功能验收

| 功能 | 验收标准 |
|------|----------|
| 定时任务 | 每周一09:00自动运行 |
| 定时任务 | 每月1号09:00自动运行 |
| 飞书通知 | 下载完成后收到通知 |
| 去重功能 | 不重复下载同一文件 |

### 4.3 日志验收

| 日志项 | 要求 |
|--------|------|
| 运行时间 | 每次运行记录 |
| 扫描邮件数 | 记录扫描数量 |
| 成功下载数 | 记录成功数 |
| 跳过数 | 记录重复数 |
| 失败数 | 记录失败数 |
| 新增文件列表 | 记录前10个文件名 |
