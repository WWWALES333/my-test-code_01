# 更新日志

## v1.2.0 - 2026-03-23

### 核心修复
- 周报周次归档修复：采用“文件名或主题周次优先，邮件日期兜底”，避免周次系统性错位
- 周次冲突审计：补充冲突记录、周月完整性摘要和审计报告输出
- 调度稳定性增强：定时任务失败不再中断守护进程，失败场景强制写入运行日志

### 稳定性优化

- IMAP 抓取重连：单封 `FETCH` 失败时自动重连并重试，优先抓取最新邮件
- 飞书通知重试：Webhook 网络抖动场景自动重试

### 数据整理
- 归档目录基线调整：仅保留 `2023` 年及以后数据
- 历史异常目录清理：移除过旧目录并收敛年份命名
